# Lookout
A collection of useful Debug Views for GameMaker.

## Views
* **Display**. Shows display, window, surface, GUI, and view information.
* **Instances**. Displays the overall and per object amounts of instances.
* **Resources**. Shows the data fetched from `debug_event("ResourceCounts")` and `debug_event("DumpMemory")`.
* **Rooms**. Provides control over room switching and displays room history.
